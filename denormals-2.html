Advanced JIT/Turbofan Float Test

This test detects whether the FPU state for "Denormals Are Zero" (DAZ) or "Flush To Zero" (FTZ) unexpectedly changes during JIT compilation (crbug.com/382005099). It iteratively multiplies a random value between 0.01 and 0.1 by 0.5 until it hits exactly 0.0. By inspecting the last non-zero value, it determines if denormals are actively used or flushed to zero. We run this 10,000,000 times in either an AudioWorklet or a Web Worker to catch if Turbofan suddenly swaps in a compiled function with a different FPU state.
